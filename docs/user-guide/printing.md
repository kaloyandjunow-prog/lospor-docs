# Protocol & Printing

After completing all three forms, LOSPOR generates a **two-page anaesthesia protocol** that can be printed or saved as a PDF.

## The two-page protocol

### Page 1 — Intraoperative

The first page contains:
- **Header** — institution name, month/year, diagnosis, patient demographics (age, sex, ASA), and a blank line for patient identity (fill in by hand after printing)
- **Intraoperative timetable** — the full graphical timetable with vital signs graph and event strip (drugs, agents, infusions, fluids)
- **Staff** — duration and blank signature lines for anaesthesiologist, surgeon, and nurse (fill in by hand)
- **Patient** — diagnosis, procedure, height/weight, BMI, IBW/ABW
- **Anaesthesia** — technique, airway device, tools, ventilation, volatile agent, position
- **Monitoring** — all active monitoring modalities
- **Vascular access** — all lines with site and size
- **Premedication** — evening and morning
- **Drug totals** — total dose of each drug administered
- **Fluid balance** — crystalloids, colloids, blood, urine output

### Page 2 — Pre- and Postoperative

The second page contains:
- **Diagnosis** and **planned procedure** (full text)
- **Risk scores** — ASA, RCRI, Apfel, STOP-BANG
- **Airway assessment** — all parameters
- **Preoperative vitals** — BP, HR, SpO₂, temperature, RR
- **Height / weight / BMI**
- **Comorbidities** — as labelled chips
- **Current medications**
- **Allergies** — including latex allergy flag
- **Laboratory results**
- **Postoperative recovery** — Aldrete score (all criteria + total), temperature, pain NRS, PONV, time in PACU
- **Disposition** — WARD / PACU / ICU with notes
- **Handover instructions** — full text of all ticked items
- **Signature lines** — anaesthesiologist, anaesthesia nurse, surgeon

## Printing

1. On the Summary page, click **🖨 Print / Save as PDF**
2. A confirmation dialog appears — verify you have reviewed the data
3. Click **Continue** — your browser's print dialog opens

:::info Patient identity
The protocol leaves patient identity fields blank. Fill in the patient's name and ID number **by hand** after printing. This information is never entered into LOSPOR or stored anywhere.
:::

4. In the print dialog:
   - Set **Paper size** to A4
   - Set **Orientation** to Landscape
   - Set **Margins** to None
   - Enable **Background graphics**
6. Click **Print** (to a printer) or **Save as PDF**

## Layout

Both pages are formatted for **A4 landscape** paper. The protocol is designed to fit exactly two pages — one sheet of A4 printed double-sided.

The timetable graph automatically scales to the duration of the case (minimum 1 hour, expanding as needed).

## Dark mode

If you are using the app in dark mode, the protocol automatically switches to a white/light background when printing, regardless of your theme setting.
